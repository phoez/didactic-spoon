## Getting Started & Testing

Repo for RockPaperScissors barebones smart contract with tests

# Infura
```sh
$ bun install
$ forge build
$ source .env
```
To run tests
```sh
$ bun test:infura
```
# Local
```sh
$ bun install
$ forge build
```
In separate terminal window
```sh
$ anvil
```
To run tests
```sh
$ bun test:local
```
